# 常见文档类型模板

## 一、调研报告

```markdown
# [主题]调研报告

## 1. 调研目标
- 明确本次调研要回答的核心问题
- 说明调研范围与限制条件

## 2. 方法与数据来源
- 采用的研究方法（文献综述/数据分析/访谈等）
- 数据来源与采集时间

## 3. 核心发现
### 3.1 发现一
- 事实描述
- 证据来源
- 影响分析

### 3.2 发现二
...

## 4. 综合分析与建议
- 基于发现的分析推论
- 可行建议及优先级排序

## 5. 不确定性与风险
- 数据局限性
- 结论的不确定程度

## 参考来源
1. [来源1]
2. [来源2]
```

## 二、对比分析

```markdown
# [A]与[B]对比分析

## 1. 对比维度
| 维度 | A | B | 说明 |
|------|---|---|------|
| ... | ... | ... | ... |

## 2. 各维度详细分析
### 2.1 [维度一]
- A 的表现：...
- B 的表现：...
- 对比结论：...

## 3. 综合评估
- A 的优势场景
- B 的优势场景
- 选择建议
```

## 三、会议纪要

```markdown
# [会议主题]会议纪要

- 日期：YYYY-MM-DD
- 参会人：...
- 记录人：...

## 议题与讨论
### 议题一：[标题]
- 讨论要点
- 决议/结论
- 待办事项（负责人 / 截止日期）

## 行动项汇总
| 序号 | 事项 | 负责人 | 截止日期 | 状态 |
|------|------|--------|----------|------|
```
