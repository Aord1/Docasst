# 事实核查与证据分级指南

## 一、证据可靠性分级

### Level 1：权威来源（可直接引用）
- 政府官方发布数据与报告
- 经过同行评审的学术论文
- 行业头部机构的一手调研报告
- 法律法规原文

### Level 2：可信来源（引用时需标注来源）
- 权威媒体深度报道
- 行业协会发布白皮书
- 知名咨询公司分析报告
- 企业官方披露信息

### Level 3：参考来源（仅作背景参考，不宜作为核心论据）
- 个人博客与自媒体文章
- 百科类开放编辑平台
- 社交媒体上的观点
- 未标注来源的二次转发

### Level 4：不可靠来源（禁止引用）
- 无作者署名的匿名信息
- 与已知事实矛盾的单一来源
- 明显带有商业推广倾向的内容
- 来源不可追溯的"据传"信息

## 二、事实核查流程

1. **交叉验证**：核心事实至少需要 2 个独立来源确认
2. **时效确认**：确认信息发布时间，排除过时数据
3. **上下文检查**：确认引用内容未被断章取义
4. **利益冲突排查**：关注来源是否与结论存在利益关联

## 三、冲突信息处理规则

1. **来源优先**：Level 1 > Level 2 > Level 3
2. **时效优先**：同级别来源，取更新数据
3. **多数优先**：同级别同时效，取更多来源支持的观点
4. **标注冲突**：无法判断时，同时列出各方观点并标注"存在争议"
